Spartan Templating is a plugin that allows theme developers to create themes that have no PHP in the template code.

== Description ==



== Installation ==

1. Upload `the spartan templating folder` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Whats with the demo theme, it looks like crap =

Its an exhaustive implementation that tries to show the spartan comments needed to pull data from wordpress into the template files.
